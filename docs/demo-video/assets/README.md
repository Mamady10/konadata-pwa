# Musique de fond (optionnel)

Déposez un fichier MP3 ici :

```
background-music.mp3
```

Il sera utilisé automatiquement par `npm run build:demo-video`.

Sans fichier, une ambiance corporate est générée automatiquement.

Sources libres de droits : [Pixabay Music](https://pixabay.com/music/), [Free Music Archive](https://freemusicarchive.org/)
